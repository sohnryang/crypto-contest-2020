## 문제 목록

### 고전 암호
- Hill cipher. 저번보다 어려움
- 행렬 연산 이용. 연립방정식 해결?

### 암호 구현
- ARX 암호 최적화 문제. 아두이노에서 빠르게 실행되어야 함.
  - loop unrolling, key scheduling 등의 최적화 기법 사용할 예정
  - 최종 단계 최적화: 어셈블리 코드 사용
- 문서화 필요, time-consumer로 추정

### 블록 암호
- padding validation code가 제공됨 (블랙박스임)
  - 윈도우 DLL 사용, VM 필요할듯
